# next-node-toolkits
> Node toolkits based on next.

[![license][license-image]][license-url]
[![size][size-image]][size-url]

## installation
```bash
yarn add @jswork/next-node-toolkits
```

## usage
```js
import '@jswork/next-node-toolkits';

nx.NodeToolkits.create();
```
